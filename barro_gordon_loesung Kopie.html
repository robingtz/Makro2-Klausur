Wohlfahrtsoptimale Kombinationen & natürliche AL-Quote

Die Aufgabe verlangt, alle kurzfristig wohlfahrtsoptimalen Kombinationen aus πₜ und uₜ in ein Diagramm einzuzeichnen, sowie die natürliche Arbeitslosenquote uₙ.

Herleitung

Die Wohlfahrtsgerade

Das wohlfahrtsoptimale Verhältnis πₜ / uₜ = 3/2 lässt sich umstellen zu:

πₜ = (3/2) · uₜ

Das ist eine Ursprungsgerade mit der Steigung 3/2. Jeder Punkt auf dieser Geraden stellt eine kurzfristig optimale Abwägung zwischen Inflation und Arbeitslosigkeit dar.

Ökonomische Interpretation: Die Zentralbank ist bereit, für eine Senkung der Arbeitslosigkeit um 1 Prozentpunkt eine Inflationserhöhung von 1,5 Prozentpunkten in Kauf zu nehmen. Je steiler die Gerade, desto inflationstoleranter ist die Zentralbank – sie gewichtet Arbeitslosigkeit relativ stärker in ihrer Verlustfunktion.

Phillipskurve t = 1 und Punkt A

Nun soll die kurzfristige Phillipskurve für Periode t = 1 eingezeichnet und der von der Zentralbank gewählte Punkt A bestimmt werden.

Schritt 1 – Phillipskurve aufstellen

Phillipskurve für t = 1

Mit π₁ᵉ = 1% und uₙ = 2% einsetzen:

π₁ = π₁ᵉ − (u₁ − uₙ)
π₁ = 1% − (u₁ − 2%)
π₁ = 3% − u₁

Das ist eine fallende Gerade mit Steigung −1, die die y-Achse bei 3% schneidet und die x-Achse bei 3%.

Warum fällt die Phillipskurve? Die kurzfristige Phillipskurve zeigt den Trade-off: Will die Zentralbank die Arbeitslosigkeit unter uₙ senken, muss sie eine Überraschungsinflation (π > πᵉ) erzeugen. Umgekehrt führt eine Inflation unter den Erwartungen zu höherer Arbeitslosigkeit.
Schritt 2 – Schnittpunkt berechnen

Punkt A: Wohlfahrtsgerade ∩ Phillipskurve

Die Zentralbank wählt den Punkt, an dem die Phillipskurve die Wohlfahrtsgerade schneidet:

// Gleichsetzen:
(3/2) · u₁ = 3% − u₁

// Auflösen nach u₁:
(3/2)u₁ + u₁ = 3%
(5/2) · u₁ = 3%
u₁ = 6/5 = 1,2%

// π₁ berechnen:
π₁ = (3/2) · 1,2% = 1,8%
Ergebnis

A Punkt A = (u₁ = 1,2% ; π₁ = 1,8%)

Die Zentralbank senkt die Arbeitslosigkeit unter uₙ = 2% auf nur 1,2%, erzeugt dafür aber eine Inflation von 1,8%, die über den Erwartungen (1%) liegt. Dies ist eine Überraschungsinflation – die Zentralbank weicht von den Erwartungen ab, um kurzfristig die Arbeitslosigkeit zu reduzieren.

Phillipskurve t = 2 und Punkt B

Die Erwartungen passen sich adaptiv an: Die Akteure beobachten, dass die tatsächliche Inflation in t=1 bei 1,8% lag, und erwarten für t=2 genau diesen Wert.

Schritt 1 – Neue Erwartungen

Adaptive Erwartungen

π₂ᵉ = π₁ = 1,8%

Die Phillipskurve verschiebt sich nach oben, weil die Inflationserwartungen gestiegen sind.

Schritt 2 – Neue Phillipskurve

Phillipskurve für t = 2

π₂ = 1,8% − (u₂ − 2%)
π₂ = 3,8% − u₂
Schritt 3 – Schnittpunkt berechnen

Punkt B: Wohlfahrtsgerade ∩ PC₂

(3/2) · u₂ = 3,8% − u₂
(5/2) · u₂ = 3,8%
u₂ = 1,52%

π₂ = (3/2) · 1,52% = 2,28%
Ergebnis

B Punkt B = (u₂ = 1,52% ; π₂ = 2,28%)

Erneut erzeugt die Zentralbank eine Überraschungsinflation (2,28% > 1,8% = πᵉ₂). Die Arbeitslosigkeit liegt wieder unter uₙ, aber die Inflation ist weiter gestiegen. Das zeigt die Inflationsspirale: Jede Runde erhöhter Inflation führt zu höheren Erwartungen, die die Phillipskurve weiter nach oben schieben.

Muster erkennen: Von A nach B steigen sowohl Inflation (1,8% → 2,28%) als auch Arbeitslosigkeit (1,2% → 1,52%). Die Zentralbank „gewinnt" in jeder Runde weniger hinzu – der kurzfristige Vorteil der Überraschungsinflation schrumpft, während die Kosten (höhere Inflation) steigen.

Mittelfristiges Gleichgewicht – Punkt C

Im mittelfristigen Gleichgewicht werden die Erwartungen erfüllt: Es gibt keine Überraschungsinflation mehr, und die Akteure haben keinen Grund, ihre Erwartungen weiter anzupassen.

Bedingung

Definition: Mittelfristiges Gleichgewicht

Im mittelfristigen Gleichgewicht gilt:

πₜ = πₜᵉ // Erwartungen werden erfüllt

Setzt man dies in die Phillipskurve ein:

πₜ = πₜᵉ − (uₜ − uₙ)
πₜ = πₜ − (uₜ − uₙ)
0 = −(uₜ − uₙ)
⇒ uₜ = uₙ = 2%
Zentrale Erkenntnis: Im mittelfristigen Gleichgewicht liegt die Arbeitslosigkeit immer bei der natürlichen Rate uₙ. Die Überraschungsinflation kann die Arbeitslosigkeit nur kurzfristig unter uₙ drücken. Mittelfristig kehrt sie auf uₙ zurück – mit dem „Preis" einer dauerhaft höheren Inflation.
Berechnung

Punkt C auf der Wohlfahrtsgeraden bei uₜ = uₙ

Da die Zentralbank weiterhin wohlfahrtsoptimal agiert und uₜ = 2%:

πₜ = (3/2) · uₙ = (3/2) · 2% = 3%

Die zugehörige Phillipskurve mit πᵉ = 3%:

π = 3% − (u − 2%) = 5% − u
Ergebnis

C Punkt C = (uₜ = 2% ; πₜ = 3%)

Das ist das strategische Gleichgewicht (Inflationsbias). Die Zentralbank hat keinen Anreiz mehr abzuweichen, weil die Erwartungen genau der tatsächlichen Inflation entsprechen. Allerdings zahlt die Gesellschaft den Preis einer Inflation von 3%, ohne dass die Arbeitslosigkeit unter uₙ gesunken ist – ein reiner Wohlfahrtsverlust.

Aufgabe e) · 4 Punkte

Glaubwürdige Politik für π = 0 – Punkt D

Nun soll die Zentralbank ein mittelfristig stabiles Gleichgewicht mit πₜ = 0 erreichen. Dafür muss sie von der Wohlfahrtsgeraden abweichen können.

Analyse

Welche Politik ist nötig?

Die Zentralbank muss eine Regelbindung (rules-based policy) ankündigen, bei der sie sich glaubhaft verpflichtet, ausschließlich Preisstabilität zu verfolgen. Konkret heißt das:

Zentralbank setzt: πₜ = 0 für alle t
// Die Zentralbank gewichtet nur Inflation, nicht Arbeitslosigkeit
// Äquivalent zu: πₜ/uₜ = 0 (statt 3/2)
Wirkungsmechanismus

Warum funktioniert das?

Wenn die Ankündigung glaubwürdig ist, passen die privaten Akteure ihre Erwartungen an:

πᵉ = 0 // Akteure glauben der Zentralbank

// Phillipskurve wird zu:
π = 0 − (u − 2%) = 2% − u

// Zentralbank setzt π = 0:
0 = 2% − u ⇒ u = 2% = uₙ
Ergebnis

D Punkt D = (uₜ = 2% ; πₜ = 0%)

Die Gesellschaft erreicht Preisstabilität bei der natürlichen Arbeitslosenquote. Punkt D ist besser als Punkt C: Beide haben u = 2%, aber D hat π = 0% statt 3%. Der Wohlfahrtsverlust ist geringer!

Kernaussage des Barro-Gordon Modells: Diskretionäre Geldpolitik (Wohlfahrtsoptimierung in jeder Periode) führt zum Inflationsbias in C. Eine glaubwürdige Regelbindung vermeidet diesen und erreicht das bessere Ergebnis D. Das Argument ist ein starkes Plädoyer für unabhängige Zentralbanken mit klarem Inflationsziel – wie die EZB.
Klausurtipp – Glaubwürdigkeit: Die Regelbindung muss glaubwürdig sein. Wenn die Akteure vermuten, dass die Zentralbank doch abweichen könnte, setzen sie πᵉ > 0 und das Gleichgewicht D bricht zusammen. Instrumente der Glaubwürdigkeit: gesetzliche Unabhängigkeit, Inflationsziel im Mandat, konservativer Zentralbankchef (Rogoff-Lösung).
